# CardputerWheel







https://github.com/user-attachments/assets/0a4ba5cd-f4b2-40c8-a545-aab4e1374e99






Controls:

Tilt Left / Right - Steering (X-axis)

; (Up Arrow) - Decrease sensitivity

. (Down Arrow) - Increase sensitivity

1 - Gamepad Button 4

2 - Gamepad Button 2

q - Gamepad Button 3

` - Gamepad Button 1

Arduino IDE Setup:

In Tools menu set:
USB Mode: USB-OTG (TinyUSB)
